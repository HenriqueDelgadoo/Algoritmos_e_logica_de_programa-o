# 💻 Algoritmos e Lógica de Programação (C e C++)

Repositório dedicado ao estudo e prática de **algoritmos e lógica de programação** utilizando as linguagens **C** e **C++**.

## 🚀 Objetivo

Este repositório tem como objetivo:

* Desenvolver o raciocínio lógico
* Aprimorar habilidades em programação
* Resolver problemas clássicos de algoritmos
* Praticar exercícios de plataformas como o Beecrowd
* Preparação para entrevistas técnicas e competições

---

## 🧠 Conteúdos abordados

* Estruturas condicionais (`if`, `else`)
* Estruturas de repetição (`for`, `while`)
* Vetores e matrizes
* Strings
* Funções
* Ponteiros (C)
* Estruturas de dados básicas
* Resolução de problemas
